# 🛍️ AI Shopping Assistant Agent

An AI-powered shopping assistant that helps users make smarter purchasing decisions using Large Language Models (LLMs). The application provides personalized product recommendations, compares products, summarizes customer reviews, and generates pros & cons using real-time product data.

## 🚀 Live Demo
https://shopping-assistant-frontend-bma0.onrender.com

## ✨ Features
- 🤖 AI-powered product recommendations
- 🔍 Natural language product search
- ⚖️ Product comparison
- ⭐ Customer review summarization
- 👍 Pros & Cons generation
- 🌐 Real-time product search using SerpAPI
- 📱 Responsive user interface
- ⚡ FastAPI REST API
- 🗄️ MongoDB database integration
- 🧠 Powered by Groq Llama 3.3 LLM

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- FastAPI
- MongoDB
- SerpAPI
- Groq Llama 3.3 LLM
